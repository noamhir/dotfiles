# Developer Context & Standards

You are assisting a developer working in a GitHub Codespaces environment.

## Project Architecture
- The primary focus is building single-page Web Applications (SPA).
- Stack: React, Vite, TailwindCSS.
- Build Target: Applications are often compiled into a single portable HTML file using `vite-plugin-singlefile`.

## Coding Standards
1. Provide literal, complete code execution. Fix what is asked without unprompted refactoring of surrounding code.
2. Ensure high stability and zero regressions.
3. Assume a modern Node environment but do not introduce new dependencies unless strictly necessary or explicitly requested.
